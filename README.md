#  DemoApp

Project created with Expo-cli

## Install

```
    yarn install
```

## iOS

```
    cd ios
    pod install
    cd ..
    yarn ios
```

## Android

```
    yarn android
```

## Tests

```
    yarn test
```

## Test users

There are 3 predefined users that you can use to test the app.

* username: Test1 // password: test1
* username: Test2 // password: test2
* username: Test3 // password: test3
